# 고배당·저변동 포트폴리오 운용 연구

배당수익률만 높은 종목을 고르면 거래가 어렵거나 실적의 질이 낮은 종목이 함께 포함될 수 있다고 생각했습니다. 그래서 KOSPI 종목을 대상으로 유동성, 배당, DPS, CFOA, 저변동성 조건을 순서대로 적용하고 실제 운용을 고려한 포트폴리오를 구성했습니다.

## 전략 구성

- 최근 거래대금을 기준으로 KOSPI 유동성 상위 500종목을 먼저 선정했습니다.
- 배당수익률이 0인 종목을 제외하고 DPS와 CFOA 조건을 적용했습니다.
- 마지막으로 고유변동성(IVOL)이 낮은 20종목을 동일가중으로 편입했습니다.
- 월별로 종목을 다시 선정하고 one-way turnover에 40bp 거래비용을 반영했습니다.

단순 백테스트 수익률만 보는 대신 월별 turnover, 평균 보유기간, AUM별 ADV 참여율, Historical Simulation VaR와 ES도 함께 계산했습니다. 운용가능규모 분석에서는 주문을 3일에 나누어 집행하고, 수수료·거래세와 거래대금 대비 market impact를 반영했습니다.

## 확인한 결과

노트북에 저장된 2021년 4월부터 2026년 4월까지의 비교 결과는 다음과 같습니다.

| 구분 | 전략(40bp 비용 반영) | KOSPI |
|---|---:|---:|
| 누적수익률 | 193.61% | 115.58% |
| CAGR | 23.60% | 16.31% |
| 연환산 변동성 | 14.74% | 29.00% |
| Sharpe | 1.52 | 0.66 |
| MDD | -14.87% | -34.64% |

이 결과는 특정 기간의 역사적 시뮬레이션입니다. 저변동성과 배당 조건이 해당 구간에서 하락폭을 줄이는 데 도움이 되었지만, 다른 시장 국면에서도 같은 성과가 반복된다고 볼 수는 없습니다.

## 구현하면서 확인한 점

수익률이 좋아도 거래대금이 작은 종목에 비중이 몰리면 실제 운용이 어렵습니다. 그래서 포트폴리오 성과와 별도로 종목별 거래대금, 예상 주문금액, 참여율을 계산했습니다. 또한 최근 한 달 성과가 장기 백테스트와 다르게 움직일 수 있어 월별 구성종목과 기여도를 따로 확인했습니다.

## 데이터와 재실행

원천 주가, 배당, 재무, 거래대금 데이터는 라이선스 문제로 저장소에 포함하지 않았습니다. 노트북에는 분석 코드와 당시 실행 결과가 남아 있으며, 전체 재실행을 위해서는 동일한 형태의 입력 파일을 `data/`에 준비해야 합니다.
